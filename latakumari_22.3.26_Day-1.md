To solve the "Remove Duplicates from Sorted Array" problem, I used a two-pointer approach to modify the array in-place. The algorithm maintains a slow pointer, i, to track the position of the last unique element, and a fast pointer, j, to iterate through the rest of the array. As j scans forward, whenever it encounters an element different from the one at i, i is incremented and the new unique element is copied to nums[i]. This process continues until the entire array is traversed, effectively shifting all unique elements to the front. Finally, returning i + 1 provides the new length of the array without duplicates.
